◆ availableKernels

exageostat::common::availableKernels
static

Set denoting the available kernels supported in matrix generation.

This set is updated automatically to add new kernels. The set is initialized with a lambda function that iterates through a directory and extracts the kernel names from the filenames. It also adds lowercase versions of the kernel names with underscores before each capital letter.

Returns
set of all available kernels names
