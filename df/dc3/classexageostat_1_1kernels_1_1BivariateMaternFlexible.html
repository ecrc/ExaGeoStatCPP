Detailed Description

template<typename T>
class exageostat::kernels::BivariateMaternFlexible< T >

A class represents a Bivariate Matern Flexible kernel.

This class represents a Bivariate Matern Flexible, which is a subclass of the Kernel class. It provides a method for generating a covariance matrix using a set of input locations and kernel parameters.
