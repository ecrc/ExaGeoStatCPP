Detailed Description

template<typename T>
class exageostat::helpers::BasselFunction< T >

The BasselFunction class provides methods for computing various derivatives of the modified Bessel function of the second kind, ( K_{\nu} ). This class is templated to support both float and double data types, enabling precision-based computations as required by different applications.

Template Parameters
TData Type: float or double
