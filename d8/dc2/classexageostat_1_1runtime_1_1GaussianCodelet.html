Transforms data from a Gaussian distribution to a non-Gaussian distribution.

Parameters
[in,out]apDescriptorZA pointer to the array of data to be transformed. This array is modified in place.
[in]apLocalThetaA pointer to the array of transformation parameters. The first element is the mean (xi), the second element is the scale (omega), the third element is the skewness (g), and the fourth element is the kurtosis (h).
[in]aSizeThe size of the data array (pZ) and the transformation parameters array (apLocalTheta).
Exceptions
std::runtime_errorIf the kurtosis parameter (h) is negative, indicating an invalid transformation parameter.
Returns
void
