Solves one of the matrix equations op( A )*X = alpha*B, or X*op( A ) = alpha*B.

Parameters
[in]aSideSpecifies whether op(A) appears on the left or on the right of X.
[in]aUpperLowerSpecifies whether the matrix A is upper triangular or lower triangular.
[in]aTransSpecifies the form of op( A ) to be used in the matrix multiplication.
[in]aDiagSpecifies whether or not A is unit triangular.
[in]aAlphaSpecifies the scalar alpha. When alpha is zero, A is not referenced and B need not be set before entry.
[in]apAThe triangular matrix A.
[in]apCDAdditional matrix CD.
[in]apCrkAdditional matrix Crk.
[in,out]apZThe matrix B of dimension, on exit is overwritten by the solution matrix X.
[in]aMaxRankMaximum rank parameter.
Returns
void
